# TrendScope Dashboard

Аналитический дашборд для системы анализа общественных трендов Казахстана.

## Стек

- **React 18** + **Vite**
- **Recharts** — графики и визуализации
- **React Router v6** — навигация
- **date-fns** — форматирование дат

## Запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить в dev-режиме
npm run dev
```

Дашборд откроется на http://localhost:5173

Backend должен работать на http://localhost:8000
Vite автоматически проксирует /api → http://localhost:8000

## Страницы

| Страница | Путь | Описание |
|---|---|---|
| Обзор | / | Общая сводка, топ трендов, timeline |
| Темы | /topics | LDA-темы с документами |
| Тренды | /trends | Временная динамика и рейтинг |
| Тональность | /sentiment | Sentiment analysis |
| Документы | /documents | Все публикации с фильтрами |
| Источники | /sources | RSS-источники и управление |

## Сборка для продакшна

```bash
npm run build
```

Статические файлы окажутся в папке `dist/`.
